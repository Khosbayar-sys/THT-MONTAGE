Politique de confidentialité

Dernière mise à jour : Juin 2025.

1. Informations personnelles recueillies

Lorsque vous visitez le site https://tht-montage.netlify.app/services, nous recueillons automatiquement certaines informations sur votre appareil, notamment :

  • Type de navigateur, adresse IP, fuseau horaire
  • Pages consultées, source de trafic, interactions sur le site

Ces informations sont appelées « Informations sur l’appareil ».

2. Technologies de collecte

Nous utilisons les technologies suivantes :

  • Cookies (_session_id, _secure_session_id, storefront_digest)
  • Fichiers journaux (adresse IP, navigateur, FAI, pages, horodatage)
  • Balises/pixels pour suivre l’activité de navigation

3. Utilisation des données

Les données collectées sont utilisées pour :

  • Traiter les commandes et paiements
  • Envoyer confirmations et factures
  • Analyser les risques et fraudes
  • Améliorer le site et son contenu
  • Vous proposer des offres selon vos préférences

4. Partage de vos données

Nous partageons vos données avec :

  • Google Analytics

5. Publicité ciblée

Vous pouvez désactiver la publicité ciblée via ces liens :

  • Facebook
  • Google
  • Microsoft Bing

Ou via https://optout.aboutads.info.

6. Ne pas suivre

Notre site ne modifie pas ses pratiques de collecte en réponse aux signaux « Do Not Track ».

7. Vos droits (Résidents européens)

Vous pouvez demander à :

  • Consulter, modifier ou supprimer vos données
  • Nous contacter à tht.montage67@gmail.com

8. Conservation des données

Nous conservons les informations de commande sauf si vous demandez leur suppression.

9. Mineurs

Le site n’est pas destiné aux personnes de moins de 16 ans.

10. Modifications

Nous pouvons modifier cette politique à tout moment. Veuillez consulter cette page régulièrement.

11. Nous contacter

Pour toute question ou demande, contactez-nous :

  • Email : tht.montage67@gmail.com
  • Adresse : 9 Rue Paul Émile Victor, VENDENHEIM, 67550, France
